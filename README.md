This is a simple contacts manager built with SvelteKit, SQLite, and better-sqlite3. It can add, view and delete contacts.

**Vercel deployment doesn't work properly at the moment so please run it locally for the full experience.**
Instructions for running locally:
1. Clone the repository by using git clone
2. Install dependencies by using 'npm install'
3. Start the development server by running 'npm run dev'
4. The app will be available at your localhost
